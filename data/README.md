# Data Directory

This directory should contain your input data files.

## Required Files

- `households.xlsx` - Household attributes and locations
- `recovery_monitoring.xlsx` - Building recovery timeline
- `shelters.xlsx` - Shelter locations
- `schools.xlsx` - School locations
- `hospitals.xlsx` - Hospital locations
- `new_buildings.xlsx` - Newly constructed buildings
- `new_jobs.xlsx` - New employment opportunities
- `new_buildings_recovery.xlsx` - Recovery timeline for new buildings
